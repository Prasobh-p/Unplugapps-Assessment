# Sales Entry Assessment

Single page Sales Entry application built using React and Redux Toolkit.

## Features
- Header section for voucher details
- Detail table with dynamic add/remove rows
- Item dropdown from API
- Auto total calculation
- Save data to backend API
- Printable voucher

## Tech
React, Redux Toolkit, Axios, Vite

## Run
npm install  
npm run dev

## Notes
Use a new voucher number when saving.
